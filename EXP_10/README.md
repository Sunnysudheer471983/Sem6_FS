# EXP 10 - Trading Journal

A simple trading journal app for tracking daily gains, losses, mistakes, and review notes.

## Features

- Add trades with symbol, entry/exit, quantity, direction, and profit/loss
- Record mistakes and lessons learned for every trade
- View daily performance summary and metrics
- Store data locally using browser localStorage
- Inspired by trading journal platforms like TraderSync, Edgewonk, and journal review workflows

## Run

1. Install dependencies: `npm install`
2. Start the app: `npm run dev`
3. Open the URL shown by Vite in your browser
